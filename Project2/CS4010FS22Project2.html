Project 2. (Optional) Online Shopping

(Due 12/19/2022 Mon)
 

Description

In this project, you will practice some more features of the servlet technology and the JSP technology. In order to reduce the development time, a sample project is provided. Download the zip file JSTL_Tutorial.zip under Part F of Module 5 for all the materials needed for this project. You need to modify the code to sell different products online.

Requirements

Basically you just implement all the features as in the sample project. You do not have to add any new feature. But the products that you sell online should be different from those in the sample project.

  1. You need to design your own database for your products. You create and populate your database by making a SQL script similar to the file bookstoredb.sql included in JSTL_Tutorial.zip. When you submit the project, this database script should be included in your project folder.

  2. To speedup the project development, you just adopt the structure of the sample project with appropriate modifications on JavaBeans, servlets, JSPs, etc. In our real-world web project development, it is very common to modify the existing code to implement the required features, which can improve our productivity.

  3. Due to the time limit in this semester, this project is set as optional. It is worth 3% extra credit (overall) if you are willing to pay the extra effort. It is quite doable, because I have seen that a few students can complete it in relatively short time.
